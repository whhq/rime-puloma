# 普逻码输入法

普逻码编码简单易记，如“普”字编码为pur，r是部首“日”的编码。
“逻”字编码为loz，z是部首“辶”的编码，lo是双拼，也可直接打拼音luo加z，“luoz”输入逻字。
“码”字编码为map，ma是拼音，部首“石”编码是p。
